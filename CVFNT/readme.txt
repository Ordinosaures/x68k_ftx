==========================================================================
       スプライト PCG → FTX フォント PCG データコンバータ CVFNT.X
                        ver 2.01       by よっしん
==========================================================================




==========================================================================
                                   説明
==========================================================================

スプライト PCG データファイル（拡張子.SP）から、テキストプレーンフォント表
示ライブラリ FTX で使用するフォント PCG データを生成するツールです。



==========================================================================
                                 使用方法
==========================================================================

CVFNT -i [入力ファイル名] -o [出力ファイル名]

# FTX


# 解説

FTX は、
SHARP X680x0 対応のテキストプレーンフォント表示ライブラリです。


以下のような特徴があります。

1) テキストプレーン上に 8x8 または 16x16 ドットの 16 色フォントを描画可能

2) フォントデータは、スプライト PCG データから変換可能

# ディレクトリ構成

- FTX/ 以下

	テキストプレーンフォント表示ライブラリ FTX 関連ファイル。

- CVFNT/ 以下

	FTX 用のフォント PCG データコンバータ。

- SAMPLE / 以下

	サンプルコード群。
